## Q. 접미사 정렬

<details>
<summary></summary>
<div markdown="1">

문자열 s가 주어지면 s문자열의 모든 접미사를 구하고, 사전순으로 출력하는 프로그램을 작성
하세요.

</div>

## A. 풀이

접미사를 구하는 것이므로 왼쪽 앞에서부터 substring의 길이를 줄여나가며 배열 answer에 넣어주면 됩니다. 사전순 정렬을 위해 결과 반환 전에 `sort()` 해줍니다.

```javascript
function solution(s) {
  let answer = [];
  for (i = 0; i < s.length; i++) {
    answer.push(s.substring(i, s.length));
  }
  answer.sort();
  return answer;
}
console.log('kseaedu');
```
