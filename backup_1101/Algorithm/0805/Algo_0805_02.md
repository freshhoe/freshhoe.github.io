## Q. 돌다리 건너기

<details>
<summary></summary>
<div markdown="1">       
돌다리를 건널 때, 한 번에 한 칸 또는 두 칸씩 건너뛰면서 돌다리를 건널 수 있습니다. 돌다리를 통해 개울을 건너는 방법은 몇 가지인지 반환해야 합니다.
</div>
</details>

## A. 풀이
계단 오르기와 같습니다.

```javascript
function solution(n) {
    let dy = Array.from({length: n+1}, () => 0);
    dy[0] = 1, dy[1] = 2;

    for (let i = 2; i < dy.length; i++) {
        dy[i] = dy[i-1] + dy[i -2];
    }
    
    return answer = dy[n];
}
console.log(solution(7)); // 34
```
